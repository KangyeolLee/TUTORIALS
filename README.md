## ▶️ Getting Started

### Client

##### .env 설정

```bash
SERVER_API_HOST=YOUR SERVER API

```

##### 실행 및 배포

```bash
# development
$ cd client
$ npm install
$ npm run start

# deploy
$ cd client
$ npm install
$ npm run build

```



### Server

##### .env 설정

```bash
PORT=

JWT_SECRET=
GIT_OAUTH_URL=
GIT_CLIENT_ID=
GIT_CLIENT_SECRET=

CLIENT_URL=

DB_PORT=
DB_USERNAME=
DB_HOST=
DB_PASSWORD=
DB_DATABASE=
```

##### 실행 및 배포

```bash
# development
$ cd server
$ npm install
$ npm run dev

# deploy
$ npm install -g pm2
$ pm2 install typescript
$ cd server
$ npm install
$ npm run deploy
```
