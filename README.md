## 배민상회 레이아웃 클론 코딩

[배민상회](https://mart.baemin.com/) 메인 페이지 클론 코딩

### Features

- 1080px 해상도만 고려
- HTML과 CSS만 사용하여 레이아웃 위주 클론 코딩
- 메인 페이지만 구현 : 내비게이션, 링크, 버튼 등 기능 작동 X
- 트랜지션/확대/반짝임 등의 사용자 인터랙션은 모두 CSS만 활용하여 구현

### Skill Stacks

- HTML5
- CSS3
